# 安装环境  

## 进入游戏云面板
    云产品 -> 游戏云 -> 管理

## 安装基础环境  
::: tip   
如果你的服务端内已经包含安装好的Forge/NeoForge/Fabric服务端核心（含完整 libraries/ 文件夹或核心）请直接安装`纯Java环境`  
如果你的服务端内不含核心，请安装`基础端`   
:::  

1. 雨云面板：上方找到 `重装/更新游戏`   
2. 安装环境：  
   如服务端未包含核心，请重装为 `对应版本的对应核心`  
   如服务端已包含核心，请重装为 `纯Java环境`  
3. 注意如果安装的是纯环境，后续需要前往 `文件管理`->`server.properties` 文件内修改 server-port为 `雨云面板`->`NAT端口映射管理`->`对外地址:端口` 对应的 `端口`（数字）,否则无法进入服务器！！！ 


::: info  

**Java环境** 选择：  
| 游戏版本 | JAVA版本 |  
| :----: | :---: |
| 1.7-1.15.2 | Java8 |
| 1.16.5 | Java11 |
| 1.17 - 1.20.X | Java17 |
| 1.20-1.21.X | Java21 |
  
:::
